Simple Radio written on C++, And using cstdlib.

------------------------------

BEFORE RUN RADIO INSTALL:

Ubuntu & Debian:

sudo apt install mpv

Arch:

sudo pacman -S mpv

SUSE:

sudo zypper install mpv

Fedora & RedHat

sudo dnf install mpv

------------------------------

Standarts Station:

1. Вести ФМ (Vesti FM)
2. Радио Маяк (Radio Mayak)
3. Русское Радио (Russkoe Radio)
4. Радио Шансон (Radio Shanshon)
5. Хайп ФМ (Hype FM)
6. Ретро ФМ (Retro FM)
7. Дорожное Радио (Dorozhnoe Radio)
8. Наше Радио (Nashe Radio)
9. Радио Ваня (Radio Vanya)
10. The ability to set your own radio link with an mp3 stream link.
